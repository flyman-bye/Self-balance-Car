# Self-balance-Car
基于 STM32F1 的串级 PID 自平衡小车
项目简介--
本项目基于 STM32F103C8T6 实现双轮直立自平衡小车，采用 MPU6050 姿态传感器采集角度与角速度，通过互补滤波融合数据，后期调试加入积分限幅和输入死区优化PID，使用三环串级 PID（角度环 + 速度环 + 位置环）实现直立平衡、速度闭环与位置控制。支持蓝牙遥控、参数在线调试避免不断烧录减少硬件Flash寿命、超声波避障，可完整展示嵌入式运动控制、闭环算法、硬件调试能力。
技术栈--
主控：STM32F103C8T6
开发工具：Keil5
控制算法：串级 PID、互补滤波
通信方式：UART、HC-05 蓝牙透传、IIC
传感器：MPU6050（姿态）、HC-SR04（超声波避障）、
驱动：电机驱动、编码器测速驱动
上位机：uni-app 小程序（蓝牙遥控 + PID 调参）
功能清单--
MPU6050 姿态采集 + 互补滤波数据融合
三环串级 PID 实现直立平衡 + 速度闭环 + 转向闭环
蓝牙透传：手机遥控前进 / 后退 / 左转 / 右转
蓝牙实时调参：PID 参数在线修改，无需重新烧录
超声波避障：自动检测障碍物并绕行
系统状态指示、异常保护

姿态传感器：MPU6050

其它：双轮小车底盘、电机、正交编码器、OLED
